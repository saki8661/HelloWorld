# Git 명령어

## Git 로컬 컴퓨터에 사용자 등록하기

- git config --global user.name "saki8661"
- git config --global user.name "saki8661@nave.com"

## Git 시작하기

- git init
- git add .
- git commit -m "최초 커밋"

## Github 업로드하기

- git remote add orgin 주소
- git push origin main

## 다시 업로드 법

- 소스코드 변경
- git add .
- git commit -m "변경내용적고"
- git push origin master
